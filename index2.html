Technical Documentation

Exura is a specialized platform for swapping Tibia coins and wrapping Tibia assets on the Solana blockchain.

System Overview

Exura operates on Solana's blockchain, focusing specifically on Tibia assets:

  • Swap Tibia coins with Solana tokens
  • Wrap Tibia gold and items as SPL tokens
  • Character verification through Tibia API
  • Automated in-game delivery system

Product Features

Swap Process

Step-by-step guide to the Exura swap process

Swapping Tokens for Tibia Coins

  1. Connect your Solana wallet
  2. Select the token you want to swap (SOL, USDC, etc.)
  3. Enter your Tibia character name
  4. System verifies character through Tibia API
  5. Complete the blockchain transaction
  6. Receive Tibia coins in-game from Exura's bank character

Swapping Tibia Coins for Tokens

  1. Connect your Solana wallet
  2. Select desired token to receive
  3. Enter your Tibia character name
  4. System verifies character through Tibia API
  5. Transfer Tibia coins to Exura's bank character in-game
  6. Once confirmed, receive tokens in your Solana wallet

API Reference

Integration example for Tibia-focused transactions

// Initialize Exura client
const exura = new ExuraClient({
    network: 'mainnet-beta',
    wallet: yourWallet
});

// Verify Tibia character
const character = await exura.verifyCharacter({
    name: 'YourTibiaChar',
    world: 'Antica'
});

// Perform a swap
await exura.swap({
    from: 'SOL',
    to: 'TIBIA_COINS',
    amount: 100,
    character: character.name
});

Frequently Asked Questions

What is Exura?

Exura is a decentralized platform that enables instant swapping and wrapping of game coins and cryptocurrencies on the Solana blockchain.

How secure are the transactions?

All transactions are secured by Solana's blockchain technology and our additional security measures, including smart contract audits and secure wrapping protocols.

What are the fees?

Exura charges minimal fees starting at 0.1% per swap, with additional network fees from Solana.
